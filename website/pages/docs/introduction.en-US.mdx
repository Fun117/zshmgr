# Introduction

`zshmgr` is a simple package manager for Zsh. It allows you to install, uninstall, update, and list packages.

### Dependencies

`zshmgr` depends on the following commands, so please make sure they are installed:

- [git](https://git-scm.com/downloads)
- [jq](https://jqlang.github.io/jq/download/)

## Installation

### Installation Steps

<Steps>

### Clone the `zshmgr` repository

```bash
git clone https://github.com/fun117/zshmgr.git
cd zshmgr
```

### Run the `install.sh` script to install `zshmgr`

```bash
chmod +x install.sh
./install.sh
```

</Steps>

Now, `zshmgr` is installed and available system-wide.
